Có thể các bạn đang đọc đã biết hết rồi. Nhưng cũng có những người chưa biết. Tớ muốn chia sẻ một chút kinh nghiệm về cái này. Có gì các bạn góp í và bổ sung nhé (yêu cầu bôi đen các kinh nghiệm)
1. PHím cửa sổ + D: Đưa ra màn hình ngoài cùng + Refresh
2. Phím cửa sổ + E: Mở Windows Explorer
3. Phím cửa sổ + F: Mở cửa sổ tìm kiếm
4. Phím cửa sổ + R: Mở mục RUN
5. Phím cửa sổ + L: Switch User trong Win
6. Phím cửa sổ + M: Đưa ra màn hình ngoài cùng
7. Phím cửa sổ + B: Tương tự khi ta chí chuột vào màn hình. Nghĩa là chọn Bkground hay có nghĩa là ko truy nhập vào cửa sổ nào.
8. Phím cửa sổ + ESC: Tẩy trắng một Form (Ví dụ như đang soạn thảo post lên KLNET. Ấn cái đấy là mất hết)
9. F5: Refresh(làm tươi màn hình)
10. F1: Windows Help

Còn rất nhiều. Nhưng hôm nay chỉ 10 cái thôi

Chương trình này sẽ biến mọi cái key trong window kia chỉ còn mang hình thức trưng bày cho đẹp mà thôi

Hot Keyboard Pro

Chương Trình Hot Keyboard Pro có thể nói là đáp ứng khá đầy đủ các nhu cầu Hotkey cho người sử dụng .

Sau khi cài đặt, chạy chương trình, bạn sẽ thấy giao diện :



1. Thiết lập Hotkey :

Để thiết lập Hotkey, bạn Click vào New Marco, bạn sẽ thấy 1 cửa sổ hiện ra.
a. Hotkey chạy 1 chương trình :
Để tạo 1 Hotkey chạy 1 chương trình nào đó, ở ô Action Type bạn chọn Open a File :


Sau đó bạn điền các thông số cho Hotkey .
-Phần Macro Name : Điền tên Chương Trình tạo Hotkey( hay gì cũng được, miễn sao khi nhìn vào bạn biết đó là Hotkey jì là được :])
-Phần Hotkey : Click chuột vào ô trống, sau đó nhấn Tố hợp fím quy định fím tắt .
-Comand Line : Chọn đường dẫn đến File thực thi chương trình tạo Hotkey.



Click OK để Save Marco đó.

b.Hotkey mở 1 thư mục ( Folder) :
-Phần Action Type chọn Open a Folder
-Folder To Open : Click vào ô ... để chọn đường dẫn đến thư mục đó.
-Gõ các thông số cho Hotkey ( như trên )
-Click OK để Save Macro :





c. Hotkey dùng mở 1 địa chỉ Web :
-Action Type : Chọn Launch Web Browers
-URL to Browers : Gõ tên Site muốn đặt Hotkey .
-Cũng điền thông số cho Hotkey như trên, ta sẽ được Macro :



2. Import/Export danh sách Hotkey :

Chương trình Hotkey Board Pro hổ trợ 1 chức năng rất hay, là có thể save danh sách hotkey lại thành 1 file rồi có import File đó thành danh sách Hotkey dùng cho ct .
a. Export :
-Click File ~> Export to File



-Chọn đường dẫn, đặt tên cho File đó.
-Click Ok để Save ta sẽ được 1 File Hotkey bao gồm các danh sách Hotkey đã thiết lập trước đó :



b. Import Hotkey : :
-Click File ~> Import From file



-Chọn đường dẫn đến File Hotkey , Click Ok ta sẽ được 1 danh sách Hotkey đã save :


Chương trình còn rất nhìu tính năng độc đáo, các bạn có thể tự khám phá sau ^^!.
Bạn có thể Down Hot Keyboard Pro 3.1.597 FullVersion tại Đây
