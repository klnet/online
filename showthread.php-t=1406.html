Mở Webcam

Để cửa sổ của webcam hiển thị trên màn hình

Ấn phím Printscreen <thường là bên tay phải, phía trên, bên trái phím Scroll Lock>

Chạy chương trình Paint: Start --> Programs --> Accessories --> Paint

Ctrl V <paste- cả cái màn hình vừa chụp lúc nãy sẽ hiện ra trong Paint>

Ctrl S để save cái màn hình lại thành một file ảnh <đuôi là .bmp>

Một số lưu ý:
- Nhớ vị trí mà mình save cái file đó, để lúc sau còn biết mà mở ra.

- Vì dùng Paint, nên đuôi của ảnh sẽ thành .bmp, ko hiển thị đc trên trang web với thể IMG, nên em cần đổi dạng ảnh sang .jpg hoặc .jpeg. Có thể dùng một số chương trình sửa chữa ảnh như ACDsee <biết mỗi cái này, chưa dùng cái nào #> rồi chọn chức năng Convert File Format <đại loại thế>

- Ngoài ra, để cả màn hình sẽ rất to, bạn đôi khi chỉ muốn mỗi hình ảnh của webcam, hãy làm các bước sau:
Sau khi Ctrl V để dán cả màn hình vào Paint --> bấm vào rồi bấm lại vào --> khoanh vùng hình ảnh webcam cần chọn --> Ctrl C --> Ctrl N <tạo mới>, chương trình sẽ hỏi bạn có muốn save cái đang làm lại ko, chọn No --> Ctrl V <bây h bạn đã có một cái ảnh webcam mà ko phải cả màn hình --> tiếp tục Ctrl S để save lại.

Chúc thành công.

Để cửa sổ của webcam hiển thị trên màn hình

Ấn phím Printscreen <thường là bên tay phải, phía trên, bên trái phím Scroll Lock>

Chạy chương trình Paint: Start --> Programs --> Accessories --> Paint

Ctrl V <paste- cả cái màn hình vừa chụp lúc nãy sẽ hiện ra trong Paint>

Ctrl S để save cái màn hình lại thành một file ảnh <đuôi là .bmp>

Một số lưu ý:
- Nhớ vị trí mà mình save cái file đó, để lúc sau còn biết mà mở ra.

- Vì dùng Paint, nên đuôi của ảnh sẽ thành .bmp, ko hiển thị đc trên trang web với thể IMG, nên em cần đổi dạng ảnh sang .jpg hoặc .jpeg. Có thể dùng một số chương trình sửa chữa ảnh như ACDsee <biết mỗi cái này, chưa dùng cái nào #> rồi chọn chức năng Convert File Format <đại loại thế>

- Ngoài ra, để cả màn hình sẽ rất to, bạn đôi khi chỉ muốn mỗi hình ảnh của webcam, hãy làm các bước sau:
Sau khi Ctrl V để dán cả màn hình vào Paint --> bấm vào rồi bấm lại vào --> khoanh vùng hình ảnh webcam cần chọn --> Ctrl C --> Ctrl N <tạo mới>, chương trình sẽ hỏi bạn có muốn save cái đang làm lại ko, chọn No --> Ctrl V <bây h bạn đã có một cái ảnh webcam mà ko phải cả màn hình --> tiếp tục Ctrl S để save lại.

Chúc thành công.
