Dùng software là tốt nhất em ạ có chương trình vừa có khả năng bảo mật folder tốt và dễ dùng mà hoàn toàn miễn phí đó là Folder Access 2.1. Chương trình này khoảng 4.16MB, tải miễn phí tại trang http://www.topdownload.net/software/view.php?id=1146
Sau khi vào cái link ở trên thì gõ Folder Access 2.1 vào mục search, lúc nó hiện ra một số các chương trình thì chọn "Download Folder Access Freeware Software"

*Sau khi cài đặt xong, chạy chương trình. Trước hết người dùng sẽ được yêu cầu thiết lập mật khẩu. Sau khi nhập mật khẩu xong, trong cửa sổ Folder Access mình phải lập danh sách thư mục cần bảo vệ bằng cách Add và chọn thư mục để đưa vào danh sách. Cũng có thể gỡ bỏ một thư mục ra khỏi danh sách bảo vệ bằng cách chọn và bấm Romove. Chương trình cho phép bảo vệ tối đa 4 thư mục, cũng có thể khoá thư mục bằng cách click phải chuột vào thư mục cần bảo vệ và chọn Lock Folder.

Nếu bạn dịnh dạng phân vùng của bạn là NTFS thì bạn có thể dùng Security của Windows :
Đầu tiên bạn vào Tools \ Folder Options...\View
Tìm đến dòng Use single file sharing ( Recommended ) và bỏ chọn ở đố để bật thêm tab Security ở Properties
Sau đó bạn có thể remove quyền truy cập của usẻ mà bạn muốn ở thư mục mà bạn cần !
