Bản quỳen bài nì thuộc zìa 4un4me.net

Đầu tiên các bạn down chương trình cửa sổ game ( phiên bản v2 mới nhất )

Download here

Down về giải nén chạy file dxwnd.exe , cửa sổ chương trình hiện ra. Bạn chọn Edit - > Add



Màn hình hiện ra



Bạn click vào nút [...] tìm đường dẫn đến thư mục game Dot Kich ( Mặc định là C:\Program Files\VTCGame\Dot Kich\ ). Ở mục Files of type bạn chọn là All Files (*.*). Tìm và chọn file crossfire.dat sau đó click vào Open.



Bạn tích vào DirectX9, Do Not Notify on Task Change, Auto-Correct Mouse Postion rồi click vào OK



Tiếp theo bạn để nguyên cửa sổ chương trình ko được tắt đi. Vào game như bình thường.



Lúc vào game bạn sẽ thấy game đã chạy ở chế độ cửa sổ nhưng vẫn còn che hết màn hình.



Bạn đăng nhập vào game chọn sever đến phần chọn phòng thì bạn Alt + Tab ra ngoài màn hình Desktop. Click trỏ phải chọn Properties cửa sổ Display properties hiện ra bạn chọn thẻ Settings. Chỉnh lại độ phân giải của màn hình trong mục Screen resolution lúc này đang ở 800*600 tăng lên (1024*768 chẳng hạn) click vào OK.



Quay trở lại game bạn sẽ thấy cửa sổ game nhỏ lại trong 1 khoảng của màn hình, h thì bạn vừa có thể chat vừa có thể chơi CF được rồi ^^



Chúc bạn vui vẻ!
