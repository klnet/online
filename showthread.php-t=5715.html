Biến máy tính thành server “leech file” từ Rapidshare

Nếu bạn có đường truyền Internet với tốc độ “ngon ngon” 1 chút (tầm 1.5Mbps trở lên) thì có thể biến máy mình thành 1 server chạy Rapidleech, tha hồ leech file trực tiếp từ Rapidshare về. Do Rapidleech dùng code PHP nên bạn cần phải tạo 1 server hỗ trợ PHP mới chạy được. Để thực hiện điều này, không có gì đơn giản hơn ngoài việc dùng USBWebserver (UW). UW là tiện ích không cần cài đặt, bạn có thể chép lên ổ USB để sử dụng ở mọi máy. Khi chạy, UW sẽ biến máy tính bạn thành 1 server “mini” với các dịch vụ như WebServer, SMTP server, hỗ trợ MySQL, Apache, PhpAdmin.

Sau khi tải UW về, bạn giải nén file *.rar vào 1 thư mục nào đó (chẳng hạn D:\USBWebserver), rồi file USB Webserver.exe để mở giao diện chương trình.

Bạn chú ý biểu tượng các dịch vụ, nếu thấy tức là UW đang chạy. Bạn mở trình duyệt, gõ vào địa chỉ http://localhost:8080 (vì mặc định, cổng web của UW là 8080) và nhấn Enter để kiểm tra. Nếu thấy trang web như bên dưới xuất hiện thì OK.



Bước tiếp theo, bạn tải về bộ code leech file. Hiện nay, thông dụng là RapidLeech, nhưng có 1 bộ code khác của Nga “xịn” hơn nhiều mà ít người biết, đó là PHPRapidGet. Trong khi Rapidleech chỉ hỗ trợ 6 site thì PHPRapidGet đúng là “hậu sinh khả úy” với 119 site! Bạn có thể xem danh sách các site được hỗ trợ tại http://rapidgetpro.ru/download.php.

Bài viết này sẽ hướng dẫn bạn làm với cả 2 bộ code trên. Đầu tiên, bạn vào thư mục D:\USBWebserver\Root, tạo 2 thư mục mới là Rapid1 và Rapid2. Sau đó, bạn giải nén dữ liệu của PHPRapidGet (file RG7.7-v1b3.zip) vào thư mục Rapid1 và dữ liệu của RapidLeech (file Rapidleech v2.3.rar) vào thư mục Rapid2. Bạn mở trình duyệt, gõ vào địa chỉ http://localhost:8080/rapid1/index.php và http://localhost:8080/rapid2/index.php để kiểm tra. Nếu thấy 2 hình dưới xuất hiện là OK. Giờ đây, máy bạn đã sẵn sàng leech file từ RapidShare rồi đấy.





Để tốc độ tải file được nhanh hơn, bạn nên dùng Internet Download Manager.

Địa chỉ download:

USbWebserver 7.0 (12.5MB): http://usbwebserver.com/UsbWebserver.rar

RapidGetPro v1.0 beta 3 (525KB): http://rapidgetpro.ru/RG7.7-v1b3.zip

RapidLeech v2.3 build 12.03.08(120KB): http://tinyurl.com/3wznw4
