Bước 2 : Chạy secman.exe trên pc , chọn next , đánh dấu vào ô Find metroTRK port automatically rồi next tiếp , nó sẽ install vào máy của bạn chương trình secman ( khỏi cần phải lằng nhằng xem port rồi chỉnh port )
Bước 3 : Mở secman trên điện thoại lên
Mở option chọn :
Turn Plat security OFF : lúc này bạn đã có thể truy cập vào các thư mục private , sys ..
Install Root certificate : install cái này lên để cài các phần mềm unsigned từ BinPDA
