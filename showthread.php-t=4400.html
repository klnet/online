Tốc độ download fụ thuộc vào n` cái chứ, muốn tăng thì chỉ có dùng soft. Phổ biến là IDM + Flashget, IDM thì toàn diện hơn, flashget thì down mms là best thì fải.

Ngòai ra thì có :

- Host: Tùy host VN hay host nước ngoài, host vn thường nhanh hơn, nhất là một số web của game online(AU, Boom,...) giờ hỗ trợ cả từng đường truyền riêng nữa, tốc độ down fải gấp 4 - 5 lần các web khác.

-Gói cước dùng: cái này thì khỏi bàn, vấn đề là các cước chỉ khác nhau khi download về, còn duyệt web thì tốc độ ko hơn nhau là mấy.
