sẵn đọc bài Lisa , một kinh nghiệm dành cho những người đang sử dụng Yahoo Chat:
- Đề phòng trường hợp nick bị hack pass hoặc bạn quên pass dẫn đến việc bạn phải làm nick mới và ngồi nhớ add lại từng người một, bây h bạn hãy vào trong hòm thư Yahoo của mình, copy cái Address Book ra thành một File và lưu trong máy tính. Trong trường hợp bạn phải tạo nick mới thì sau khi tạo xong chỉ việc vào nick mới và update cái Address Book. Sau đấy bạn chỉ cần vào Yahoo Messenger, chọn Tab Address Book, click chuột vào nick bạn muốn add, rồi chọn Add to Messenger List.

Cách để copy Address Book thành File như sau:
- Login vào tài khoản Yahoo của bạn.
- Vào Mail Option ( góc trên bên phải )
- Vào Address Book ( bên tay trái )
- Vào Import/Export ( bên tay phải )
- Trong phần Export, chọn Yahoo! CVS: Export Now ( cái nào cũng đc )
- Sau đấy nó hiền ra một cửa sổ hỏi bạn muốn Save vào máy ko, bạn chỉ việc chọn Save rồi đièn tên và chọn chỗ lưu trữ trong máy tính. Bạn đã tạo xong file lưu giữ Address Book.

Nếu bạn muốn Update cái Address Book đó vào nick mới của bạn, bạn cũng vào phần Import/Export , nhưng sau đấy chọn trong phần Import:
1 -> chuyển sang Yahoo! (.CVS file )
3 -> chọn Browse rồi tìm trong máy tính file mà bạn đã tạo.
Cuối cùng bạn ấn Import Now ( 4 ) , Address Book của nick này sẽ giống hệt Address Book của nick cũ của ban.
