Đây là kỹ thuật giấu file đơn giản mà hiệu quả.
Đầu tiên vào window explorer tạo một cái thư mục tên gì cũng đc nhưng tốt nhất là cứ đặt Recycled để khỏi thu hút sự chú ý.

Ví dụ với folder "Test recycle" -> trước khi xử lí:
bên trong:

Sau đó tạo 1 file tên desktop.ini trong đó, nội dung như sau:
[hide=3]
Code:
[.ShellClassInfo]
CLSID={645FF040-5081-101B-9F08-00AA002F954E}
[/hide]
- Save file đó lại và set thuộc tính ẩn rồi chép các file cần dấu vào, có thể tạo thêm thư mục bên trong đó.
- Mở Command Prompt lên hoặc Start -> Run rồi gõ attrib +S +H <folder name>
Foldername là cái thư mục recycled (hoặc thư mục bạn đã tạo)

- refresh lại -> biểu tượng thư mục biến thành thùng rác, click vào trong không thấy gì, empty recycled bin không sao file vẫn còn, các file vẫn tuy xuất được bình thường nếu biết address file.
Kết quả như sau:
bên trong:

* Khi nào cần xem thì làm như trên: Mở Command Prompt hoặc Start -> Run attrib -S -H <folder name>

-> Lại như ban đầu:
