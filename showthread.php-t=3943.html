(2x+y)2 - 5( 4x2 -y2 ) + 6(2x-y)2 = 0 (pt1)

3y = (y2 +2) / x2 (pt 1)
3x = ( x2+2)/y2 ( pt2)

x+y-V(xy) = 3
V(x+1) + V ( y+1) = 4

Hệ IV

x2 + y2+x+y = 4
x(x+y+1 ) + y(y+1) = 2

V(2x+y+1) -V(x+y) =1
