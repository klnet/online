Dupe Checker PRO cho phép bạn tìm tất cả những bản sao của file và folder rồi xóa chúng đi. Dupe Checker PRO sẽ xác định tất cả file và folder trùng lặp có tên giống nhau, những file bị nhân đôi, nội dung bị trùng lặp và có cùng dung lượng. Dupe checker PRO cũng cho phép bạn tìm tất cả những file rỗng và folder rỗng.

Tính Năng:

* Liệt kê những file khác thường
* Xem trước những file dạng ảnh,HTML và text.
* Hiện tất cả file và folder với những bản sao chính xác ở bất cứ đâu trên nhiều ổ đĩa hệ thống (Ổ Cứng, Flash, Floppy, USB, Ổ Jaz và nhiều ổ đĩa mạng dạng ảnh)
* Cung cấp những công cụ giúp xóa và di chuyển những file trùng lặp 1 cách an toàn và dễ dàng.
* So sánh các file từng byte một.
* File Name Match: tìm kiếm những tên file trung lặp với nội dung khác nhau.
* Size Match: chọn tên file, ngày giờ chỉnh đã chỉnh sửa và dung lượng file.
* Những file rỗng không có nội dung. Những file có dung lượng bằng 0.
* Bộ lọc mạnh với nhiều lựa chọn lọc
* Bộ lọc mạnh đã được thiết lập trước cho file Internet, file Microsoft Office, file Ảnh, file Nhạc và Phim.
* Bộ lọc được cá nhân hóa có thể lưu lại và sử dụng nhiều lần.
* Lựa chọn màu cho từng nhóm trùng lặp. Sắp xếp theo Folder trùng lặp, Nội Dung trùng lặp và Tên trùng lặp.
* Làm Việc với nhiều ổ đĩa và những ổ đĩa được truy cập qua mạng.
* Hỗ trợ đầy đủ cho định dạng phân vùng FAT, FAT-32 và NTFS.
* Hiện dung lượng theo byte, KB, MB hoặc GB.
* Giao diện sử dụng theo phong cách Windows Exporer thân thiện và dễ sử dụng.
