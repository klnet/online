1. Đầu tiên là mở stock ra. Filter-->Sharpen-->Sharpen (có thể bỏ qua nếu hình đủ nét rồi )
2. Layer-->Duplicate Layer 2 lần cái stock, để Soft Light cả 2 layer vừa dup lên, giữ nguyên opacity 100%. Sau đó Image-->Adjustments-->Desaturate layer trên cùng:

3. Layer --> New fill layer --> Solid Color --> OK. Fill với code màu là #113067. Để Exclusion với opacity khoảng 90-100%:

4. Thêm 1 fill layer với code màu #bce6f2. Color burn 100%:

5. Lại 1 fill layer, code #ffd991. Soft light 90-100%:

6. Fill layer, code #fc9191. Soft light 100%:

7. Fill layer, code #f8dc9e. Multiply 30-40%:

8. Duplicate layer đã desaturated ở bước 2. Drag nó lên trên cùng:

9. Layer-->New Adjustment Layer-->Curves-->OK. Click đại vào chỗ nào đấy để 2 ô thông số hiện lên, type vào Input: 129, Output: 126:

10. Layers-->New Adjustment Layer-->Colour Balance. Ở pop up hiện ra nhớ chỉnh phần Color thành Yellow-->OK. Type vào 3 ô số: -15, -11, +14:

Sau 1 loạt các thể loại thì sẽ được những layer như này:

11. Flatten image/Merge all layer. Vậy là xong Tiếp theo là add text và brush loạn xạ j đấy tuỳ thik
