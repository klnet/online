đơn giản nhứt là xách cái đĩa hiren zìa ,zô dos quét qua 1 phát , del mấy cái fiel ngớ gẩn của em kavo đi,roài zô Win quét = bkav,restart lại roài kill tiếp = tay
Trình tự zư này nè:
Cách 1

1) Chạy Task Manager (Ctrl-Alt-Del)(rút hết usb ra nhá)
2) Nếu wscript.exe đang chạy~> kill
3) kill EXPLORER.EXE
4) run ~> cmd
5)type autorun.inf (xem mấy cái file nóa chạy đóa , nhớ tên để sau kill)
6) attrib -s -h -r autorun.inf(bỏ đi trạng thái file ẩn , chỉ đọc và file system,những cái thể lại nài hem cho xóa )
del autorun.inf
(trong cái file autorun.inf cóa tên file gì thì kill hết = 2 dòng trên)
7) Tới thư mục Windows\System32 bằng cách chạy lệnh "cd c:\windows\system32" (không có dấu nháy kép)
8) Chạy lệnh "dir /a avp*.*
9) Nếu bạn thấy các files có tên dạng avp0.dll hay avpo.exe hay avp0.exe hay kavo.exe hay kavo0.dll hay kav0.exe hay kav0.dll (túm lại là kav gì đó), dùng lệnh sau để xóa từng file một:

attrib -r -s -h avpo.exe
del avpo.exe

10) Dùng run từ Task Manager để chạy regedit
11) Tìm đến HKEY_CURRENT_USER \ SOFTWARE \ Microsoft \ Windows \ CurrentVersion \ Run
12) Tìm lại tất cả những tên file giốn trên (trừ thèng explorer.exe nhá),click đúp zô, kéo đến cuối ,thấy tên nóa ở đâu thì xóa cái tên đó đi (đừng xóa cả cái key của nóa nhá,nguy hiểm lắm ,nhàm lả khỏi bật máy lun đóa )
13) Khởi động lại máy
14)Copy file cmd.exe trong C:\Windows\system32\ vừa từng ổ roài zô Run ,gõ [Tên ổ]:\cmd.exe roài kill đóng autorun.inf trong các ổ đó.~> khởi động lại nhát nữa
15)Cài Bkav roài quét full 1 lần ,h thì chắc là yên tâm


Cách 2

Sử dụng đĩa Hiren Boot CD, cách này chỉ áp dụng cho ổ đĩa phân vùng FAT32:

1) Cho đĩa vào ổ CD, khởi động lại máy, chú ý đặt thiết lập BIOS cho máy boot bằng đĩa CD
2) Chạy Mini Windows
3) Truy cập vào từng ổ đĩa bằng cách: vào My Computer, nhấn chuột phải vào từng ổ chọn Open
4) Xóa files ntde1ect.com và autorun.inf
5) Đi tới thư mục windows>system32 trong ổ đĩa chứa windows bị virus.
6) Tìm và xóa files apv0.dll, apvo.exe và apv0.exe (nếu có)
7) Bỏ đĩa CD ra khỏi ổ, khởi động lại máy, vào windows, chạy regedit
8) Tìm tới HKEY_CURRENT_USER \ SOFTWARE \ Microsoft \ Windows \ CurrentVersion \ Run
9) Xóa avpo.exe nếu có
10)Dùng mấy cái antiVirus trong đó (M-Affe với cái khỉ gì ý,quên mất roài) quét phát,nhớ là quét full nhá,hem chơi quét kiểu nhanh gọn thuận tiện đâu đấy )
Nếu ổ cứng của bạn ở định dạng NTFS, hãy chạy Paragon Mount Everything (support NTFS) trong đĩa Hirent Boot và thực hiện xóa các files kể trên.
=========================
