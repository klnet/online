DigitSoft DiskShop v3.2.1828


Quản lý file và folder và chạy các tiện ích ổ đĩa với hệ thống 3 góc nhìn. Bên cạnh đó còn có một chức năng thiết lập trước và chức năng dọn dẹp ổ đĩa. Các công cụ chỉnh sửa bao gồm một chức năng sửa tên cải tiến và bộ lọc cải tiến cùng chức năng xem các thuộc tính và ngày tháng.

Bạn có thể dò tìm history của folder. Bạn cũng có thể dò tìm các mục lựa chọn trong tất cả các folder. Bên cạnh đó còn có thểm chức năng Favorite cho các file và các fovorite của Internet. Nó cũng bao gồm chức năng dọn dẹp và tìm kiếm với các lệnh của người dùng và nút Go. Bạn có thể xem các thuộc tính của file và folder cung như xem trực tiếp các ảnh.

Chương trình hỗ trợ nhiều skin. Một màn hình desktop ảo được tích hợp cùng các chức năng gián đoạn. Chương trình cho phép đánh giá dung lượng trống của ổ đĩa và cảnh báo dung lượng thấp khi copy hoặc di chuyển dữ liệu. Bạn có thể chuyển đổi giữa loại thời gian 12 giờ hoặc 24 giờ. Nó bao gồm dòng lệnh cho file, chức năng tìm file với hỗ trợ ký tự thay thế và chức năng tìm folder. Chương trình cũng có khả năng tìm kiếm file theo dung lượng và thuộc tính.

Các Điểm Nổi Bật Của DiskShop:
Hệ thống 3 góc nhìn đồng thời.
Chế độ xem 2 Panel hoặc một Panel với chế độ hình cây. Bạn có thể copy, di chuyển và tạo folder trong giữa các panel.
Quản lý File và Folder nhanh tuyệt đối nhờ hệ thống 3 panel của DiskShop.
Khung TreeView (một panel) được xem như là nguồn lệnh thứ ba cho việc chọn lựa file và folder trong các Panel.
Tổng hợp thông tin và bảo mật khu quản lý file và folder trong ổ đĩa của bạn.
Chương trình được tối ưu về dung lượng và tốc độ.
Di chuyển và thay đổi kích thước chương trình trên màn hình - lưu các vị trí.
Truy cập folder của bạn với 1 cú click từ history folder và lưu history của bạn vào file để tự động load.
Truy cập file của bạn từ "danh sách Recent file" với 1 cú click từ history file và có thể quay lại tới 25 file.
Các phím nóng bàn phím cho tất cả các lệnh quan trọng.
Các khung ảnh trong mỗi panel cho phép xem các ảnh số của bạn ... (có khả năng tắt bật tính năng này).
Tạo file, Chỉnh sửa file và truy cập vào MS-Dos
Sắp xếp các file và folder của bạn từ chế độ panel.
Thay đổi tên file và folder, đơn lẻ hoặc theo nhóm, cùng với các lệnh để liệt kê và Icase/Ucase.
Điều khiển tất cả các thuộc tính file và folder, đơn lẻ hoặc theo nhóm.
Khởi động các chương trình bằng cách click đúp file thực thi trong các panel.
Duyệt tất cả các chi tiết file và folder quan trọng trong cửa sổ riêng biệt, đơn lẻ hoặc theo nhóm.
Chức năng tìm kiếm và dọn dẹp ổ đĩa có thể lập trình với khả năng mở file từ các chế độ hiển thị cộng với các lựa chọn xóa file riêng lẻ.
Sử dụng thiết lập có sẵn để truy cấp các vị trí folder ưa thích một cách nhanh chóng.
Bên cạnh các thiết lập có sẵn bạn có thể tạo các lựa chọn ưa thích như các favorite của file và Internet.
Chức năng memory cho các chế độ xem đã chọn (memory store, memory recall và undo recall)
Mở menu địa chỉ File, Folder, Internet. Lưu trữ 15 mục gần nhất.
Truy cập mạng Lan và duyệt menu ổ đĩa.
Duyệt Ổ Đĩa của bạn với chức năng history.
Chức năng dòng lệnh tích hợp được dành để khởi động các chương trình cải tiến.
Tối ưu tốc độ mạng của bạn.
Xem thông tin Ổ Đĩa và Ram.
Chức năng hoán đổi các Panel.
So sánh và đồng bộ hóa các folder với cửa sổ xem trước.
Màn hình desktop ảo.
Kéo và thả giữa các chế độ xem.
Copy tới hoặc copy từ desktop.
Tính năng lọc chế độ hiển thị cải tiến với chế độ xem theo khoảng thời gian, các chỉnh sửa người dùng và các thuộc tính của file.
Tính năng lọc file cải tiến với các chức năng giống như tính năng lọc hiển thị ở trên.
Tính năng dò tìm và quay lại nhanh các lựa chọn trước đó trong tất cả các folder.
Thay đổi màu sắc các mục trong mỗi panel để xác định chế độ bạn đang sử dụng, copy, di chuyển hay xóa.
Chuyển đổi giữa hệ thống đồng hồ 12 giờ hoặc 24 giờ.
Dòng lệnh tìm kiếm file nhanh để truy cập file nhanh trong khung hiển thị (cho phép các ký tự thay thế)
Tìm nhanh các folder và menu.
Duyệt các folder đích từ bên trong các chức năng copy/move với khả năng "return"
Duyệt 3D từ menu phải chuột (folder trong suốt)
Lọc,Dọn Dẹp,Tìm kiếm với các thuộc tính kích thước file với các lựa chọn khoảng thời gian.
Chức năng Zip file
Chức năng so sánh file theo dòng với cách hiển thị trực quan.
Tích hợ chức năng Copy As và Move As.
Chức năng xuất ra file Exel.
Hỗ trợ cho Windows 98, -Me , -XP và Vista.
Nâng cấp và hỗ trợ trong thời gian sử dụng sản phẩm.
