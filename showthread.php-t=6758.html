Tiếp tục series bypazz win vào chơi CF ,h tớ giới thịu 1 CD khá đặc biệt : miniPE
Các bạn có thể download bản iso của nó ở đây :
Code:
http://minipe.org/
Ngoài công dụng chính đề cập trong bài viết nài là bypazz win ,Cd nài là 1 đĩa cứu hộ ,có khả năng diệt virus , giúp bạn backup dữ liệu, ghi đĩa ,khôi phục dữ liệu bị xoá trước đó , bla bla bla các phần mềm tiện ích khác

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
H thì bắt đầu lào
B0: check quả phát ,h chúng ta có 2 user : XzobinX ( admin ) và Bjn siu nhan ( limited user )

B1 : nhét đĩa vào ổ ,khởi động lại,đĩa sẽ tự boot vào màn hình chính của nó

B2 : chọn Program~>System tool ~> Password Renew 1.1 Beta

B3 : Chọn đường dẽn đến thư mục cài Win

Đến đây các bạn có 4 sự lựa chọn
B4.1 : Đổi pazz của user (trò nài trêu sếp vs trường phòng hay đừng hỏi )

Chọn Install

Done
