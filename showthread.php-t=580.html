Đầu tiên bạn vô Windows Explorer . Tiếp đó bạn vô View >>> chọn Folder Options >>>chọn Custom based on settings you choose: >>> chọn tiếp settings >>>> chọn For all forders width HTML content rồi bạn chọn OK.
Xong bước đầu bạn thoát khỏi Windows Explorer rồi vào lại. Lần này bạn vào tiếp thư mục View >>>> chọn Customize this Folder... rồi bạn chọn Create or edit an HTML document >>>>> rồi bạn ấn vào Next cho đến khi thấy xuất hiện một cửa sổ soạn thảo bằng Notepad là được . Khi này bạn chỉ cần chèn thêm đoạn script này vào trang soạn thảo đó rồi vô File >>> save lại là được rùi. Đoạn script đó nè :
<script language="JavaScript"> var pass = prompt("Enter the password"); if(pass!= "you-pass-word") {go.history(-1);} else {window.location-"C:"} </script>

*Chú ý : bạn hãy thay password mà bạn muốn đặt vô chỗ Enter the Password nhé.
Cuối cùng nó hiện ra một bảng bạn chỉ cần Finish là xong. Xong rồi bây giờ thoát ra thử lại nhé, bạn thử mở một thư mục nào ra xem nào . Hì, đòi pass ngay đó bạn ạ !
