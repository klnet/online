Làm theo những bước sau đây:

1) Download the game iso, mount and install with daemon tools.

DO NOT RUN THE GAME BEFORE DOING THE FOLLOWING STEPS!

2) Download and run the patch

3) Download and open the crack files part 1

4) Extract these files to your clearsky\bin folder (Backup the older files first if you feel the need)

5) Download and open crack files part 2 (protect.dll)

6) In your "clearsky\bin" folder, rename the original protect.dll to protectxxx.dll

7) Copy the cracked protect.dll (from crack files part 2) to the "clearsky\bin" folder

8) Ensure the image is still mounted in daemon tools

9) Run clear sky from either the exe file or the desktop shortcut.

10) A message will appear to attempt to verify your disk, and will ask about installing some software. Click yes and let it run

11) It should crash with a bugtrap error. Close this window.

12) Unmount the clear sky image from daemon tools

13) Delete the cracked protect.dll and rename the original from protectxxx.dll back to protect.dll

14) Download the english language db file.

15) Copy "xenglish.db" into your "clearsky\localization" folder. Delete the russian original file.

16) Run the game from your desktop shortcut and enjoy!
